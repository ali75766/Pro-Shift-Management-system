# Pro Shift Management System

Pro Shift Management System is a web-based application developed to manage employee shifts and attendance efficiently.

## Project Overview
This project helps organizations manage employee attendance records, shift schedules, and related management tasks through an easy-to-use interface.

## Technologies Used
- Angular
- .NET
- SQL Server

## Features
- Employee Attendance Management
- Shift Scheduling
- User Management
- Data Management

## Installation & Setup

1. Clone this repository
2. Install dependencies
3. Run the Angular application:

ng serve
